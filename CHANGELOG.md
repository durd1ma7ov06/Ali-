- 🦾 ESP32 servo control (head tracking + arm movements)

- 🎯 AI-powered movement planner
- 🔒 Security policy (SECURITY.md)
- 📋 Contributing guide (CONTRIBUTING.md)

- 📖 Setup documentation (docs/SETUP.md)

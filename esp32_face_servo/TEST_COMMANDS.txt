SERIAL MONITOR SOZLAMALARI
=================================================================

Arduino IDE Serial Monitor:
- Baud rate: 115200
- Line ending: Newline (yoki Both NL & CR)

Putty:
- Speed: 115200
- Connection type: Serial
- Implicit CR in every LF: Yes

Python (Serial Monitor):
- import serial
- ser = serial.Serial('COM3', 115200)
- ser.write(b'HEAD:90\n')
- print(ser.readline())


=================================================================
TROUBLESHOOTING
=================================================================

Agar komandalar ishlamasa:

1. ✅ ESP32 tayyor signal berganmi?
   Kutiladi: "ESP32_HUMANOID_SERVO_READY"

2. ✅ Baud rate to'g'rimi?
   115200 bo'lishi kerak

3. ✅ Line ending to'g'rimi?
   Newline (\n) yoki Both NL & CR

4. ✅ Port to'g'ri ochilganmi?
   Device Manager da COM port ko'rish

5. ✅ Servolarga power berilganmi?
   5V, 3A external adapter

6. ✅ Ground umumiyligini?
   ESP32 GND = Power Supply GND


=================================================================
QISQA YORDAM
=================================================================

HEAD:<0-180>              Boshni burish
ARMS:<6 ta offset>        Qo'llarni boshqarish  
SET:<id>:<angle>          Bitta servo
HOME                      Neutral pozitsiya
STATUS                    Hozirgi holat
SMOOTH:<ON|OFF>           Yumshoq harakat
CALIBRATE:<id>:<angle>    Kalibrlash
HELP                      Yordam

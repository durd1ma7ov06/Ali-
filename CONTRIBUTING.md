# 🤝 Contributing to Ali Robot

Rahmat! "Ali" AI robot loyihasiga hissa qo'shmoqchi bo'lganingiz uchun minnatdormiz.

## 🚀 Qanday hissa qo'shish mumkin

### 1. Fork qiling
Loyihani fork qilib, o'z branchingizda ishlang.

### 2. Branch yarating
```bash
git checkout -b feature/yangi-funksiya
```

### 3. O'zgarishlarni kiriting
- Kodni yozing yoki xatolikni tuzating
- Test qo'shing (iloji bo'lsa)
- Commit qiling: `git commit -m "feat: yangi funksiya qo'shildi"`

### 4. Pull Request oching
- PR ochganingizda, nima o'zgartirganingizni yozing
- Screenshot yoki video qo'shing (iloji bo'lsa)

## 📋 Hissa qo'shish yo'nalishlari

| Yo'nalish | Tavsif |
|-----------|--------|
| 🐛 Bug fix | Xatoliklarni topib tuzatish |
| 🌐 Tarjima | Interfeys tarjimalarini qo'shish |
| 🤖 Yangi harakatlar | Robot uchun yangi harakat buyruqlari |
| 📚 Documentation | Hujjatlarni yaxshilash |
| 🧪 Tests | Test coverage'ni oshirish |
| 🎨 3D/Avatar | 3D model va avatar yaxshilashlari |
| 🧠 AI/NLP | So'zlashuv va NLP yaxshilashlari |

## 🔧 Texnik talablar

- Python 3.10+
- Flake8 linting xatoliklarsiz
- `.env.example` yangilangan bo'lishi kerak
